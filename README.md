# 📁 File Explorer

A simple command-line File Explorer built with Python and pathlib.

---

## 📖 Description

- This is a simple command-line file explorer.
- It allows users to browse, inspect, and manage files and folders.
- I built this project to learn more about the `pathlib` module.
- It helped me practice file and folder operations using Python.

---

## ✨ Features

- Select a folder
- Show all items
- Show only files
- Show only folders
- Count files
- Count folders
- Show file information
- Show extension statistics
- Show folder summary

---

## 🛠 Technologies

- Python 3
- pathlib (Python Standard Library)

---

## 📂 Project Structure

```text
project/
│
├── main.py
├── explorer.py
└── README.md
```

- `main.py` → Where users interact with the file explorer.
- `explorer.py` → Contains all file explorer functions.
- `README.md` → Project documentation.

---

## 🚀 How to Run

1. Clone the repository.

2. Open the project folder.

3. Run:

```bash
python main.py
```
